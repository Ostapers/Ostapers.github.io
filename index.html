About

Ostap parser-combinator library now in OPAM. Svn history has not been saved. Autotools abandoned, OCamlBuild is used now.

OPAM 
opam remote add ostap http://ostapers.github.io/opam
opam update ostap
opam install ostap

How to test everything without adding opam repo 
      opam pin add typeutil https://github.com/Ostapers/typeutil.git
      ...
      opam pin add checked  https://github.com/Ostapers/checked.git
      ...
      opam pin add ostap    https://github.com/Ostapers/ostap.git
      ...
      opam install ostap -t -v
