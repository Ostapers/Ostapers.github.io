About

Ostap parser-combinator library now in OPAM. Svn history has not been saved. Autotools abandoned, OCamlBuild is used now.

OPAM 
opam remote add ostap http://ostapers.github.io/opam
opam update ostap
opam install ostap

How to test everything without adding opam repo 
opam pin add logger  https://github.com/Kakadu/logger-1.git    -n -y
opam pin add GT      https://github.com/Kakadu/GT.git          -n -y
opam pin add ostap   https://github.com/Kakadu/ostap.git       -n -y
opam pin add ocanren git+https://github.com/Kakadu/miniKanren-ocaml-typed.git\#upstream-ob -n -y
opam install ostap   -t -v   # with tests and verbose output
opam install ocanren -t -v
